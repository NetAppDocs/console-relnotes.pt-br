=== Atualizações de nomes de recursos de planejamento de ciclo de vida

Os seguintes recursos de planejamento do ciclo de vida tiveram seus nomes alterados:

* O recurso de atualização tecnológica agora se chama *Atualização do sistema*. Saiba como  https://docs.netapp.com/us-en/console-lifecycle-planning/use/tech-refresh.html["avaliar uma atualização do sistema"].
* O recurso de Adição de Capacidade agora se chama *Expansão de Armazenamento*. Saiba como  https://docs.netapp.com/us-en/console-lifecycle-planning/use/capacity-add.html["expandir armazenamento"].
* O recurso de Renovação agora se chama *Support Renewals*. Saiba como  https://docs.netapp.com/us-en/console-lifecycle-planning/support/task-support-registration.html["Cadastre-se para suporte"].
